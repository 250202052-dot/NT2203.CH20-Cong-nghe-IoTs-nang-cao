# NT2203.CH20 - Công nghệ IoT nâng cao

## 📌 Giới thiệu
Repo này được xây dựng nhằm phục vụ môn học **Công nghệ IoT nâng cao**, bao gồm:
- Các bài tập thực hành trên Google Colab.
- Tài liệu tham khảo về bảo mật hệ thống, mã độc nâng cao.
- Ứng dụng thử nghiệm với ThingSpeak IoT.
- Mô hình huấn luyện và dự đoán dữ liệu IoT.

## 📂 Nội dung chính
- **Colab Notebooks**
  - `Mã độc nâng cao.ipynb`: Thực hành phân tích mã độc nâng cao.
  - `TraningModel.ipynb`: Huấn luyện mô hình học máy cho dữ liệu IoT.
  - `Predict_model_iot.ipynb`: Dự đoán dữ liệu IoT bằng mô hình đã huấn luyện.

- **Tài liệu tham khảo**
  - [Buffer Overflow - Viblo](https://viblo.asia/p/buffer-overflow-in-a-nutshell-m68Z0ox6KkG)
  - [Hugging Face Models](https://huggingface.co/models) – nguồn mô hình AI phục vụ nghiên cứu.

- **Công cụ & Nền tảng**
  - [ThingSpeak IoT](https://thingspeak.mathworks.com) – nền tảng thu thập và phân tích dữ liệu IoT.
  - [Kali Linux](https://www.kali.org/get-kali) – hệ điều hành phục vụ nghiên cứu bảo mật.
